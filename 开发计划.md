## Excel甘特图制作说明

### 方法一：使用Excel堆叠条形图创建甘特图

1. **准备数据**：
   - 在Excel中创建以下列：任务名称、开始日期、持续时间（天数）
   - 计算"开始日期数值"：使用公式将日期转换为数值（例如：`=B2-DATE(2025,12,26)+1`，其中B2是开始日期）

2. **创建图表**：
   - 选择数据区域（任务名称、开始日期数值、持续时间）
   - 插入 → 图表 → 条形图 → 堆叠条形图
   - 设置图表格式：
     - 将"开始日期数值"系列设置为无填充（透明）
     - 将"持续时间"系列设置为有颜色的填充
     - 反转类别顺序（使第一个任务在顶部）
     - 设置日期轴格式

3. **美化图表**：
   - 添加图表标题："项目开发计划甘特图"
   - 设置坐标轴标签
   - 添加图例
   - 调整颜色和样式

### 方法二：使用Excel模板

参考链接：https://blog.coupler.io/gantt-chart-excel-template/

按照该模板的步骤创建甘特图。

### 方法三：使用CSV数据文件

项目已提供 `甘特图数据.csv` 文件，可以直接在Excel中打开使用。

详细制作步骤请参考 `Excel甘特图制作指南.md` 文件。

## 将甘特图插入文档

### 方法一：插入Excel图表对象（推荐）

1. 在Word文档中，将光标定位到要插入甘特图的位置
2. 点击"插入" → "对象" → "由文件创建"
3. 选择包含甘特图的Excel文件
4. 勾选"链接到文件"（可选，这样Excel文件更新时，Word中的图表也会更新）
5. 点击"确定"

### 方法二：复制粘贴图片

1. 在Excel中，选中甘特图
2. 右键点击图表，选择"复制"
3. 在Word文档中，将光标定位到要插入的位置
4. 右键点击，选择"粘贴"或"选择性粘贴" → "图片（增强型图元文件）"

### 方法三：另存为图片后插入

1. 在Excel中，右键点击甘特图
2. 选择"另存为图片"
3. 保存为PNG或JPG格式
4. 在Word文档中，点击"插入" → "图片" → "此设备"
5. 选择保存的图片文件

### 方法四：使用截图工具

1. 在Excel中显示完整的甘特图
2. 使用截图工具（如Windows的Snipping Tool）截取图表
3. 在Word文档中插入截图

完成后的甘特图应包含以下要素：

1. **图表标题**：项目开发计划甘特图（2025.12.26-2026.01.13）
2. **Y轴（垂直轴）**：任务名称列表

   - UI界面模块
   - 业务逻辑模块

   - 文件导出模块
   - 集成测试与优化
   - 文档编写
3. **X轴（水平轴）**：日期范围（2025-12-26 至 2026-01-13）
4. **任务条形**：每个任务用彩色条形表示
   - 条形的起始位置表示任务开始日期
   - 条形的长度表示任务持续时间
   - 不同任务可以使用不同颜色区分

### 任务拆分合理性说明

本开发计划的任务拆分遵循以下原则：

1. **模块化拆分**：按照系统功能模块（UI、数据库、网络、线程等）进行拆分，便于分工协作
2. **依赖关系清晰**：明确各任务之间的依赖关系，确保开发顺序合理
3. **时间分配合理**：
   - 核心模块（业务逻辑）分配4天，确保功能完整实现
   - 基础模块（数据库、UI）并行开发，提高效率
   - 辅助模块（网络、线程）可以并行开发
   - 预留缓冲时间，应对突发问题
4. **里程碑明确**：设置7个关键里程碑，便于进度跟踪和验收

## 风险控制

1. **技术风险**：
   - Qt版本兼容性问题 → 提前测试不同版本
   - 网络功能调试困难 → 准备本地测试服务器

2. **进度风险**：
   - 业务逻辑复杂 → 预留缓冲时间
   - 测试时间不足 → 边开发边测试

3. **质量风险**：
   - 代码质量 → 代码审查和重构
   - 用户体验 → 早期用户反馈
